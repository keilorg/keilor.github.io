/* #2) Scandinavia Checking a list The word IN allows us to check if an item is in a list. The example shows the name and population for the countries 'Brazil', 'Russia', 'India' and 'China'. Q. Show the name and the population for 'Sweden', 'Norway' and 'Denmark'. */
SELECT name, population
FROM world
WHERE name IN ('Sweden', 'Norway', 'Denmark');
/* #3) Just the right size Which countries are not too small and not too big? BETWEEN allows range checking (range specified is inclusive of boundary values). Q. Show the country and the area for countries with an area between 200,000 and 250,000. */ SELECT name, area FROM world WHERE area BETWEEN 200000 AND 250000;
